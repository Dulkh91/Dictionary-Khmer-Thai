## Khmer–Thai Dictionary iOS App

Personal iOS project built with SwiftUI.

### Features
- Khmer–Thai word lookup
- Simple and clean UI
- Local data handling

### Tech Stack
- Swift
- SwiftUI
- MVVM (basic)

### Purpose
This project was built to practice iOS development fundamentals and real app structure.
